#### 1. gateway.py — Discord Gateway WebSocket

Dùng thư viện websockets (lightweight, không dùng discord.py).

Flow:
GET https://discord.com/api/v10/gateway/bot → lấy WebSocket URL
Connect WebSocket
Nhận HELLO (op=10) → lấy heartbeat_interval
Gửi IDENTIFY (op=2):
{
  "op": 2,
  "d": {
    "token": "<BOT_TOKEN>",
    "intents": 33281,
    "properties": {
      "os": "linux",
      "browser": "command-listener",
      "device": "command-listener"
    }
  }
}

Intents = GUILDS (1) | GUILD_MESSAGES (512) | MESSAGE_CONTENT (32768) = 33281
Chạy heartbeat loop song song (gửi op=1 mỗi heartbeat_interval ms)
Nhận events (op=0) → dispatch MESSAGE_CREATE events
Handle reconnect: lưu session_id + sequence → RESUME (op=6) khi bị disconnect
Auto-reconnect với exponential backoff (1s → 2s → 4s → ... max 60s)

#### 2. dispatcher.py — Command routing

COMMAND_ROUTES = {
    "1509957355549495431": {  # #system-status
        "status": "commands.status.handle",
    },
}


Logic:
Nhận MESSAGE_CREATE event
Bỏ qua nếu author.bot == True (ignore bot messages, kể cả chính mình)
Check channel_id có trong COMMAND_ROUTES không
Parse content: strip prefix (/ hoặc !), lấy command name (word đầu tiên)
Nếu command match → gọi handler → post response
Nếu không match → ignore silently

#### 3. commands/status.py — Handler
 (2/5)
import asyncio
from pathlib import Path

SCRIPT = Path.home() / ".hermes" / "scripts" / "system_watchdog.py"
TIMEOUT = 10

async def handle(channel_id: str, content: str, author: dict) -> str:
    """Run system_watchdog.py --force, return stdout."""
    try:
        proc = await asyncio.create_subprocess_exec(
            "python3", str(SCRIPT), "--force",
            stdout=asyncio.subprocess.PIPE,
            stderr=asyncio.subprocess.PIPE,
        )
        out, err = await asyncio.wait_for(proc.communicate(), timeout=TIMEOUT)
    except asyncio.TimeoutError:
        return "⚠️ Status check timed out (>10s)"
    except FileNotFoundError:
        return "⚠️ Script not found: system_watchdog.py"

    if proc.returncode != 0:
        return f"⚠️ Script error (rc={proc.returncode}):\n
\n{err.decode()[:500]}\n
"

    output = out.decode().strip()
    return output or "(no output)"


#### 4. config.py — Configuration

from pathlib import Path
import re

ENV_FILE = Path.home() / ".hermes" / ".env"

def load_token() -> str:
    text = ENV_FILE.read_text()
    m = re.search(r'^DISCORD_BOT_TOKEN\s*=\s*(.+?)\s*$', text, re.MULTILINE)
    if not m:
        raise SystemExit("DISCORD_BOT_TOKEN not found in .env")
    return m.group(1).strip().strip('"').strip("'")


#### 5. main.py — Entry point

import asyncio
import logging
import signal

async def amain():
    setup_logging()
    token = load_token()
    gateway = GatewayClient(token, intents=33281)
    dispatcher = Dispatcher(gateway, COMMAND_ROUTES, post_response)
    
    # Graceful shutdown
    loop = asyncio.get_running_loop()
    stop = asyncio.Event()
    for sig in (signal.SIGTERM, signal.SIGINT):
        loop.add_signal_handler(sig, stop.set)
    
    task = asyncio.create_task(gateway.run())
    await stop.wait()
    task.cancel()

if __name__ == "__main__":
    asyncio.run(amain())


#### 6. Response posting — dùng REST API
 (3/5)
Post response qua REST (không qua WebSocket):
POST https://discord.com/api/v10/channels/{channel_id}/messages
Authorization: Bot <token>
Content-Type: application/json

{"content": "<response text>"}


Dùng httpx (async). Handle rate limit (429 → retry after retry_after).

Discord message limit = 2000 chars. Nếu output > 1900 chars → truncate + append \n...(truncated).

#### 8. install.sh — Setup script

#!/bin/bash
set -e

DIR="$(cd "$(dirname "$0")" && pwd)"
cd "$DIR"

# Create venv
if [ ! -d .venv ]; then
    uv venv .venv --python 3.11
fi

# Install deps
uv pip install -e . --python .venv/bin/python

# Install systemd user service
mkdir -p ~/.config/systemd/user
cat > ~/.config/systemd/user/command-listener.service << EOF
[Unit]
Description=Discord Command Listener
After=network-online.target
Wants=network-online.target

[Service]
Type=simple
WorkingDirectory=$DIR
ExecStart=$DIR/.venv/bin/python -m command_listener.main
Restart=always
RestartSec=5
Environment=PYTHONUNBUFFERED=1

[Install]
WantedBy=default.target
EOF

systemctl --user daemon-reload
systemctl --user enable command-listener
systemctl --user start command-listener
echo "✅ command-listener service installed and started"
